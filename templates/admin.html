Admin tools

This page is only visible to users with IDs listed in the admin section of config.toml.

  • The Observations page has an admin-only filter to view, export, and delete any user's observations.
  • The Bookings page has an admin-only filter to view and cancel upcoming bookings for any user. Admins can book an unlimited number of slots.

Adjust pointing offsets

If observations of a strong source (e.g. the Sun) show the peak at an az/el offset, enter that offset here to correct the MD01 controller's stored position. The reported position decreases by the entered values; the rotor does not move. The telescope must be in maintenance mode and not currently booked. See the ROT2PROG protocol reference for details on the calibration command.
